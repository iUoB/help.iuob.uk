## 我拿到签证了，现在需要做什么

完成 Right To Study（RTS）Check ，上传资料，访问：https://intranet.birmingham.ac.uk/student/international/myrts/guides/Guides-by-visa-type.aspx，根据你的签证类型，选择并上传已有资料

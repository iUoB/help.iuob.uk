# 如何使用伯明翰大学SSO登陆Zoom （简体中文）

---

以下以University of Birmingham为例

  1. 进入Zoom登陆界面
   
   ![](./1.png)
  
  2. 选择SSO登录
   
   ![](./2.png)
  
  3. 输入SSO域名，并点按继续
   
   ![](./3.png)
  
  4. 如果你不知道SSO域名，请使用邮箱进行查找

   ![](./4.png)
  
  5. 跳转至SSO认证界面
     
   ![](./5.png)

  6. 登陆成功即可访问[Zoom.us](https://zoom.us)账户
   
   ![](./6.png)

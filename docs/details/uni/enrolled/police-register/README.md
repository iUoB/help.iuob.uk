# 伯明翰大学入学后警察局注册相关问题

## 我需要去警局注册吗

签证如果有需要你 Police Registration ，那你就需要在伯明翰警局注册

请参考学校给出的信息: http://url.iuob.uk/UoB-Police-Reg-Info 

和

伯明翰警察局给出的信息: http://url.iuob.uk/Bham-Police-Reg-Info


## 我需要警局注册，但是我要隔离怎么办

先隔离，后拿 BRP ，再注册，不用担心时间限制
